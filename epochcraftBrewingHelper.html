Epochcraft Brewing Helper

Perfecting brew aspects

Setup

Create many cauldrons, and number them with signs. Write down in the "Cauldron notes" section of this page what cauldrons correspond to what tests, so that you can keep track of everything you're testing at once. It's recommended that you have X cauldrons per person, so that each person can work on every aspect of a different potion at once. You can open the Binary Search Helper in multiple tabs to help you test different attributes at the same time. It's recommended that one person only tries to keep track of a single brew.
Create 6-8 large barrels— one for each type— oak, birch, spruce, jungle, acacia, dark oak, crimson hyphae, warped hyphae.

Possible ingredients

Before you try perfecting any brews, figure out a list of all possible ingredients accepted by the plugin by throwing random items into a cauldron until you think you've found all possible ingredients. This is the only step you have to do before perfecting specific brews besides setting up the cauldrons and casks.

Determine how close you are to perfecting the potion

The better the potion effects are (in strength and duration), the closer you are to perfecting the recipe. Once you reach a point in perfecting an individual attribute where the values do not increase further, you have perfected that attribute.

Ingredients

See which combinations of ingredients produce a valid brew; just try every permutation. Once you have a list of valid ingredients for each brew, perfect the amounts for a brew by using the Binary Search Helper for the amount of each ingredient.

Cook time

Time you leave it cooking in a cauldron. Use the Binary Search Helper. Different brews display different cauldron bubble colors at different times, though this can be misleading and potentially completely-unrelated to their optimal cook time.

Distill runs

How many times to run the brew through a brewing stand. Some brews should not be distilled (0 distill runs). Use the Binary Search Helper. Distill brew by putting it in the bottom (where potions go) of a brewing stand and glowstone in the top (where ingredients go). Different brews take a different amount of time to distill per run (though this is not a manipulatable variable).

Age

How many Minecraft days you leave it in a barrel for. Use the Binary Search Helper.

Barrel wood type

What type of wood the barrel is made of. It is recommend that you make 6-8 copies of the same brew and put them in every different type of barrel for the same amount of time to see which wood type is correct.
